# Arcane Swarm (2D Magic Survival benzeri)

Basit bir HTML5 Canvas survival prototipi:

- Oyuncu WASD / ok tuşları ile hareket eder.
- Arcane Bolt otomatik en yakın düşmana ateş eder.
- Seviye atlayınca büyüler güçlenir, 3. seviyede Nova Ring açılır.
- Düşmanlardan XP shard düşer, toplayınca seviye atlanır.

## Çalıştırma

`index.html` dosyasını tarayıcıda açmanız yeterli.
